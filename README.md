# This is a repository for a Python + LLM + Agentc AI project

Through this repository we will learn how to integrate AI assistants, specifically Claude, in our Python coding workflow.

## What is this?
This project is a starting point for:
- Python development
- AI tools and APIs
- Prompting
- Debugging
- Building small applications with the help of AI

To use this repository, make sure Python is installed on your computer (current repo version 3.13.3 as of 2026/03/07). Go to https://www.python.org/downloads, find the desired Python version and download it for your operating system. During installation make sure the "Add Python to PATH" option is checked.

You can check Python is installed with:
```
python --version
```

To clone the repository:

```
git clone https://github.com/DimiVamv/ai-python-lab.git
cd ai-python-lab
```

We will be using the "uv" virtual environment for ease-of-use to create virtual environments.
To install it visit this page and follow the instructions: https://docs.astral.sh/uv/getting-started/installation/

TL;DR, there are a few different ways to install UV, and these are the following:

For Mac OS terminals:

```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

and if your system doesn't have curl:

```
wget -q0- https://astral.sh/uv/install.sh | sh
```

For Windows, open Windows PowerShell as administrator and run:

```
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```
With pip:

```
pip install uv
```

To install dependencies with pip or with uv:

```
pip install -r requirements.txt
```
```
uv pip install -r requirements.txt
```

Then install the local package in editable mode so imports like `from ai_python_lab...` work from examples, lessons, and tests:

```
python -m pip install -e .
```

After all requirements have been downloaded and installed, we will copy the example .env.example file into a new file named ".env", that will contain our Claude API key. We can do this manually or with the following Terminal command if we are inside our repository folder: 

```
cp .env.example .env
```

In the new ".env" file, we will replace the placeholder "your_api_key_here" with the API key you created from the Claude Workspace website.

To check everything is in working order, we will run the main script in Terminal, or simply press Run in our IDE. 
For Terminal:

```
python -m ai_python_lab.main

python examples/example_01_simple_prompt.py
```

Important rules for the lessons:
1. Do not upload your API key to GitHub. This is your private key, and only you should know it.
2. Do not share the .env file, as this contains the Claude API key. That is why we have the .gitignore in our repository, to avoid accidentally uploading it.
3. Always study and test code generated by AI. Do this with everything "AI Generated".
4. Try to understand the code before using it.
5. Have fun!
